¿Qué es display grid?

"Display grid" es una herramienta de diseño web que permite crear estructuras de cuadrícula para organizar el contenido en una página web. Es como una tabla invisible que divide la página en filas y columnas, permitiendo colocar elementos de contenido dentro de ellas.

La ventaja de usar display: grid es que es muy flexible y permite a los diseñadores web controlar exactamente dónde se colocan los elementos en la página. Esto significa que se pueden crear diseños personalizados que se adapten a las necesidades del sitio web y se vean bien en diferentes dispositivos, como computadoras de escritorio, tabletas y teléfonos móviles.

Además, display: grid ofrece un control preciso sobre el espacio y la alineación de los elementos en la página. Los diseñadores web pueden ajustar el tamaño de las filas y columnas de la cuadrícula, así como agregar espacio entre los elementos, lo que puede ayudar a mejorar la legibilidad y la claridad del contenido.

Ejemplo de código y diseño

El código HTML define un contenedor div que tiene tres elementos div con clases "elementos a", "elementos b" y "elementos c", que contienen la letra correspondiente.

HTML 
<div class="contenedor">
    <div class="elementos a">a</div>
    <div class="elementos b">b</div>
    <div class="elementos c">c</div>
</div>

El CSS define el contenedor div con un borde redondeado, un relleno y un fondo verde azulado. El contenedor está definido como una cuadrícula de 3 filas y 3 columnas con un espacio de 1 rem entre filas y columnas.

Los elementos div tienen un borde redondeado, un relleno y un fondo de color verde amarillento, con texto blanco.

La clase .a, .b, .c hace que los elementos div tengan un tamaño de fuente grande (2rem) y se alineen tanto horizontal como verticalmente en el centro utilizando la propiedad display: flex, justify-content: center y align-items: center. Las clases .a, .b y .c también tienen una propiedad de grid-row que establece la posición vertical de cada elemento dentro de la cuadrícula.

.a ocupa las filas 1 a 3, .b ocupa las filas 2 a 3 y .c ocupa solo la fila 3.

CSS 
.contenedor {
    border-radius: 0.4rem;
    padding: 1rem;
    display: grid;
    grid-gap: 1rem;
    grid-template-columns: repeat(3, 1fr);
    grid-template-rows: repeat(3, 1fr);
    background-color: hsl(0, 0%, 22%);
}
          
.elementos {
    border-radius: 0.2rem;
    padding: 1rem;
    background-color: hsla(0, 0%, 100%, 0.5);
    color: white;
}
          
.a {
    grid-row: 1 / 4;
}
          
.b {
    grid-row: 2 / 4;
}
          
.c {
    grid-row: 3 / 4;
}

El diseño quedaria así.

Un contenedor con 3 elementos, el contenedor tiene 3 filas y 3 columnas.

Cada elemento esta denotado por una letra, la letra a ocupa una columna y 3 filas, la letra b ocupa una columna y 2 filas y la letra c ocupa una columna y una fila.

En resumen

"display grid" es una herramienta poderosa y versátil que permite a los diseñadores web crear diseños de cuadrícula personalizados para organizar el contenido de una página web de manera eficiente y atractiva.
